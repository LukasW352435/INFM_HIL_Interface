REST Dummy DuT

This dummy provides the possibility to send and receive messages.

To build and execute you can use docker (optionally mount config file).

docker build -t your-image-name .
docker run -it --rm -p 9090:9090 -v $(pwd)/config:/opt/config your-image-name:ro

If you want to compile locally the following libraries are needed:

  • libasio-dev
  • libssl-dev

This implementation uses the restbed framework as well as the OpenAPI Stub Generator. Restbed is provided as git submodule, use git submodule update or git clone --recursive in order to checkout with submodules.

The generated stub (from OpenApi Spec) has to be slightly modified, see generatedAPI/README.md.
